Jade 1.0.0 - Documentation
==========================

Quick start
-----------
  jade                    Start REPL
  jade idle               Start Jade IDLE (interactive shell)
  jade file.jdl           Run a Jade file
  jade run file.jdl       Same
  jade check file.jdl     Check syntax
  jade build file.jdl     Compile to native binary (optional LLVM)
  jade jolt --help        Package manager

File extension
--------------
  .jdl  Jade source files (double-click to run with jade.exe)

Project and packages
--------------------
  jade jolt init          Create a new project (jade.toml)
  jade jolt add <pkg>     Add dependency
  jade jolt install       Install dependencies
  In your code: import mylib

For more, run: jade --help
